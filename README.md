🎁 제품 홍보 포스터 생성기
OpenAI API와 Streamlit을 활용하여 키워드만 입력하면 홍보 문구와 포스터 이미지를 자동 생성해 주는 웹 서비스입니다.

사용 스택

Python

Streamlit

OpenAI API (gpt-4o-mini, gpt-image-2)

실행 방법

라이브러리 설치: pip install -r requirements.txt

API 키 설정: .streamlit/secrets.toml 파일 생성 후 키 입력

앱 실행: streamlit run poster.py
